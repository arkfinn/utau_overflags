# OverFlagsとは？
音階によってFlagsを設定できるUTAU用プラグインとして作られました。
単に一括でFlagsを変更するツールとしても使えます。
条件に応じて様々な操作ができるプラグインとして進化中です。

--------------------------------------------------------------------------------

# 利用方法
このプラグインは.Net6.0で作られました。
グーグルなどで調べてインストールしてください。
UTAUやOpenUtauのプラグインとして使えます。

--------------------------------------------------------------------------------

# 操作方法

画面から察して下さい。

## Flagsの追記・上書き・置換・計算　について
追記は既存のFlags情報を残したまま後ろに追記、
上書きは既存のFlags情報を消した上で上書きします。

置換はFlagsに記入されたフラグが既に存在する場合も置き換えます。

- 元の設定：g10Y0
- OverFlagsの設定：g5H0
この設定の時、下のように動作します。

- →追記：g10Y0g5H0
- →上書き：g5H0
- →置換：g5Y0H0

計算は独自の表記をします。

フラグ名と数値の間に「+」「-」「%」を付けると計算をします。

- 元の設定：g10Y50
- OverFlagsの設定：g+5Y10H0
→g15Y10H0

- 元の設定：g10Y50
- OverFlagsの設定：g-5Y%50
→g5Y25

## プリセットの読み込み・追加・削除

上部プルダウンの選択でプリセットの読み込みが可能です。
この際、現在の設定値は失われますので注意してください。

プルダウン横の保存アイコンからは、現在の設定値の保存が行えます。

プリセットはMy Documents以下のAppData\Roaming\utau_overflags\preset以下に保存されます。
不要なプリセットはここから直接削除してください。

1.81まではプラグイン設置フォルダ以下のpresetフォルダに保存されていました。引き続き読み込むことができます。
1.01までのプリセットも以上いずれかに設置することで流用可能です。
